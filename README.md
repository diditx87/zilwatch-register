# ZilWatch Tools

1. Create Wallet (Private Key)
2. Send 1 Zil to Address
3. Regist Address to Zilwatch

## Installation

Use [nodejs](https://nodejs.org/en/) to Run Code.
Install Npm module
```bash
npm i
```

## Usage

```Nodejs

# Create Wallet
node create.js

# Fund 1 zil to Address after create
node send.js

# Regis wallet into Zilwatch
node regis.js
```
